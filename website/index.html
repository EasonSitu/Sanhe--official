關於我們

三禾行為食品與零食相關業務用途網站，現階段先建立公開展示頁，方便後續以 GitHub 持續維護與更新。

服務內容

產品展示

可補充品牌、產品分類、節慶禮盒與常備商品資訊。

批發合作

可放合作流程、適用客戶類型與聯絡窗口。

內容更新

後續可持續更新活動資訊、季節主推與網站公告。

產品方向

日常零食

糖果、餅乾、沖調類、休閒食品等基礎分類可逐步上站。

節慶禮盒

年節、婚慶、應節型產品可獨立做專題頁面。

聯絡方式

此區可後續補上電話、微信、電郵、地址或表單連結。

目前為基礎佔位頁，內容可按實際對外版本再補齊。
